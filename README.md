# formulario-cadastro
Um formulário de cadastro para prática de preenchimento automático via Fetch API e persistência de dados

O objetivo final é uma página contendo uma aplicação de cadastro de usuários que utilize Fetch API para preencher automaticamente o endereço a partir do CEP (usando a API do ViaCEP) e que salve os dados do formulário no Web Storage API, garantindo que as informações permaneçam preenchidas mesmo após o usuário recarregar a página.

Para este projeto serão usadas as linguagens HTML, CSS e JavaScript
